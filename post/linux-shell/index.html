Linux 調整shell 方式

2022-02-21

tags: linux

查看系統/bin/sh 目前shell

要查看/bin/sh 目前連結到哪個Shell，可以使用以下指令:

$ ls -l /bin/sh

想要切換到bash可以使用:

sudo dpkg-reconfigure dash

會看到以下畫面:

  • 選擇Yes 則/bin/sh 會連結到dash;
  • 選擇No 則/bin/sh 會連結到bash。
